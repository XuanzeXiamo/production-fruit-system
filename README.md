# 🍎 水果识别专家系统（产生式推理）

[![Python](https://img.shields.io/badge/Python-3.6+-blue.svg)](https://www.python.org/)
[![License](https://img.shields.io/badge/License-MIT-green.svg)](LICENSE)

基于产生式规则（Production Rules）的水果分类专家系统，采用正向推理机制，模拟人类专家根据水果特征进行识别决策的过程。

## 📋 项目简介

本项目是一个经典的**专家系统**实现，通过预定义的推理规则链，根据用户输入的水果特征（颜色、形状、表皮、大小等），逐步推导并识别出具体水果种类。

**核心特点：**
- 🧠 **产生式推理**：基于 IF-THEN 规则的知识表示
- 🔗 **正向链推理**：从事实出发逐步推导结论
- 🏗️ **分层推理架构**：单层特征 → 中间概念 → 最终分类
- 🎯 **7种水果识别**：苹果、香蕉、西瓜、橙子、草莓、葡萄、芒果

## 🚀 快速开始

### 环境要求
- Python 3.6+

### 运行方式
```bash
python fruit_recognition.py
```

输入对应条件前面的数字:

* 内部: 21多汁 22明显果核 23种子在表面 24可剥皮                                  *

请输入：1
请输入：7
请输入：13
请输入：19
请输入：22
请输入：29

推理过程如下：
正圆形 + 红色系 + 极光滑 -> 核果类
红色系 + 圆形/扁圆 + 极光滑 + 中小型 + 明显果核 + 核果类 -> 苹果

所识别的水果为苹果

### 🔬 技术亮点
产生式系统架构：清晰分离知识库与推理机，符合专家系统经典设计

分层推理策略：先推导中间概念（热带水果、浆果类等），再判定具体水果

规则优先级管理：通过代码顺序实现规则的优先级排序

重复检测机制：judge_repeat() 防止同一结论被多次推导
